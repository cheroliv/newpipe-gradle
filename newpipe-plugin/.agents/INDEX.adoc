== Règles Absolues (Exécutif)

=== 0. Environnement de Dev

```bash
# Depuis newpipe-plugin/ (répertoire courant) :
./gradlew :newpipe:clean :newpipe:build
./gradlew :newpipe:test
./gradlew :newpipe:functionalTest
./gradlew :newpipe:publishToMavenLocal
```

*publishToMavenLocal* obligatoire après toute modif de code source du plugin.

=== 1. Git — Interdictions

❌ `git add`, `git commit`, `git push`, `git commit --amend`
✅ `git status`, `git diff`, `git log`

=== 1b. Fichiers de Configuration — Sûreté

❌ Write écrasant de config, remplacer secrets par placeholders
✅ `git check-ignore` + `git ls-files` avant modif, backup pré-modification

=== 2. Tests en Fin de Session

❌ Lancer tests sans permission explicite

=== 3. Procédure de Fin de Session (6 étapes obligatoires)

. Archive `.agents/sessions/{N}-{type}-{sujet}.adoc`
. MAJ `PROMPT_REPRISE.adoc` (N+1, max 70)
. MAJ `SESSIONS_HISTORY.adoc`
. MAJ `INDEX.adoc`
. MAJ `TEST_COVERAGE_ANALYSIS.adoc` (si tests modifiés)
. MAJ `COMPLETED_TASKS_ARCHIVE_YYYY-MM.adoc`
. Rotation backup si N%10==0 OU EAGER > 500 lignes

*Checklist OBLIGATOIRE* en fin de session :
```
✅ Procédure de fin de session exécutée
📋 Checklist :
[✅] 1. Archive session N créée
[✅] 2. PROMPT_REPRISE.adoc mis à jour pour session N+1
[✅] 3. SESSIONS_HISTORY.adoc mis à jour
[✅] 4. INDEX.adoc mis à jour
[✅] 5. TEST_COVERAGE_ANALYSIS.adoc mis à jour (si applicable)
[✅] 6. COMPLETED_TASKS_ARCHIVE.adoc mis à jour
[✅] 7. Backup roté (si applicable)
```

=== Détail des Migrations

==== 1. LlmClient
*Source* : `mp3-organizer/src/main/kotlin/com/mp3organizer/ai/LlmClient.kt`
*Destination* : `newpipe/src/main/kotlin/com/cheroliv/newpipe/ai/LlmClient.kt`
*Adaptation* : Changer package `com.mp3organizer.ai` → `com.cheroliv.newpipe.ai`

==== 2. PlaylistAgent → SessionGeneratorAgent
*Source* : `mp3-organizer/src/main/kotlin/com/mp3organizer/ai/PlaylistAgent.kt`
*Destination* : `newpipe/src/main/kotlin/com/cheroliv/newpipe/ai/SessionGeneratorAgent.kt`
*Adaptations* :
- PostgreSQL → NewPipeExtractor
- SQL → Configuration YAML YouTube
- Contexte DB → Schéma YouTube (filtres, genres, artistes)

```kotlin
// mp3-organizer : SQL
val sql = llm.chat("SELECT * FROM tracks WHERE genre = 'jazz'")

// newpipe-gradle : YAML
val config = llm.chat("filters: { genre: jazz, decade: 1960s, instrument: piano }")
```

==== 3. PlaySmartTask → GeneratePlaylistTask
*Source* : `mp3-organizer/src/main/kotlin/com/mp3organizer/tasks/PlaySmartTask.kt`
*Destination* : `newpipe/src/main/kotlin/com/cheroliv/newpipe/tasks/GeneratePlaylistTask.kt`
*Adaptations* : DB query → YouTube search via NewPipeExtractor, XSPF → Playlist YouTube, supprimer VLC

==== 4. Version Catalog
*Source* : `mp3-organizer/gradle/libs.versions.toml`
*Destination* : `newpipe-gradle/gradle/libs.versions.toml`

```toml
[versions]
kotlin = "2.3.20"
gradle = "9.4.1"
langchain4j = "1.12.2"
newpipe-extractor = "0.24.0"

[libraries]
langchain4j-core = { module = "dev.langchain4j:langchain4j", version.ref = "langchain4j" }
langchain4j-ollama = { module = "dev.langchain4j:langchain4j-ollama", version.ref = "langchain4j" }
newpipe-extractor = { module = "com.github.TeamNewPipe:NewPipeExtractor", version.ref = "newpipe-extractor" }
```

==== 5. Config YAML
*Inspiration* : `plantuml-gradle/plantuml-plugin/src/main/kotlin/plantuml/PlantumlConfig.kt`

```kotlin
data class NewpipeConfig(
    val llm: LlmConfig,
```

== EPIC — Download with NewPipe while Authenticated on YouTube

*Objectif* : Permettre le téléchargement de vidéos YouTube en étant authentifié avec des comptes Google :
- Accéder aux vidéos réservées aux membres
- Contourner les restrictions géographiques (geo-blocked)
- Accéder aux vidéos avec restriction d'âge (age-restricted)
- Éviter les limitations de taux (rate limiting)
- Télécharger des playlists privées

=== US-1 — Authentification Initiale des Comptes Google ✅
*Priorité* : 🔴 Haute | *Points* : 3 | *Date* : 2026-04-14

*En tant que* utilisateur du plugin
*Je veux* authentifier mes comptes Google via OAuth2 Device Flow
*Afin de* pouvoir télécharger des vidéos réservées aux membres

*Critères d'acceptation* :
```bash
./gradlew authSessions
# Affiche un code à 12 chiffres et une URL
# L'utilisateur va sur https://google.com/device
# Il entre le code et accepte les permissions
# Résultat : sessions.yml mis à jour avec refreshToken
# Token jamais affiché dans les logs
# Support multi-comptes
```

*Fichiers* : `AuthSessionTask.kt`, `SessionManager.kt`, `Models.kt`

=== US-2 — Téléchargement avec Sessions Authentifiées ✅
*Priorité* : 🔴 Haute | *Points* : 5 | *Date* : 2026-04-14

*En tant que* utilisateur authentifié
*Je veux* que le téléchargement utilise automatiquement mes sessions
*Afin de* télécharger des vidéos réservées aux membres

*Critères* :
- SessionManager distribue les requêtes en Round-Robin
- Vidéos membres téléchargées
- Fallback anonyme si toutes sessions échouent

*Fichiers* : `DownloadMusicTask.kt`, `DownloaderImpl.kt`, `SessionManager.kt`

=== US-3 — Refresh Automatique des Tokens Expirés ✅
*Priorité* : 🟠 Moyenne | *Points* : 5 | *Date* : 2026-04-15

*En tant que* utilisateur du plugin
*Je veux* que les tokens d'accès soient rafraîchis automatiquement
*Afin de* ne pas avoir d'interruption pendant les téléchargements

*Comportement* :
. Vérifier si `accessToken` est expiré
. Si oui, refresh via `refreshToken` → nouvel `accessToken`
. Si refresh échoue (HTTP 400), marquer session invalide
. Retry avec autre session

*Fichiers* : `TokenRefresher.kt` (créé), `SessionManager.kt` (modifié), `NewpipeManager.kt` (modifié)
*Tests* : 27 tests unitaires passants

=== US-4 — Gestion des Erreurs d'Authentification ✅
*Priorité* : 🟠 Moyenne | *Points* : 3 | *Date* : 2026-04-15

*En tant que* utilisateur du plugin
*Je veux* être informé clairement des problèmes d'authentification
*Afin de* pouvoir les résoudre rapidement

*Scénarios d'erreur* :
. refreshToken révoqué → « Exécutez `./gradlew authSessions` »
. refreshToken expiré → idem
. client_secret invalide → « Vérifiez `client_secrets/...` »
. quota API dépassé → bascule automatique sur autre compte
. compte Google suspendu → session désactivée

*Fichiers* : `AuthErrorHandler.kt` (créé), `SessionManager.kt`, `DownloaderImpl.kt`, `DownloadMusicTask.kt`
*Tests* : 17 tests AuthErrorHandler + 5 tests SessionManager ajoutés

=== US-5 — Vidéos 18+ avec Vérification d'Âge ✅
